# STROYKLIMAT SaaS Strategy
## Витрина HVAC для України

---

## 📊 Executive Summary

**STROYKLIMAT** — це комплексна платформа-витрина для HVAC (вентиляція, кондиціонування, опалення) магазинів України. Розроблена для інтеграції з Prom.ua та іншими маркетплейсами, надаючи власникам магазинів:

- ✅ Профілю каталогу з 1250+ товарів
- ✅ Адміністративної панелі для управління каталогом
- ✅ Режиму «Витрини» (запрос цены замість покупки)
- ✅ Запірливих форм запиту цін через WhatsApp, Viber, Telegram
- ✅ Повної інтеграції з системами CRM та мессенджерами

**Ціль**: Стати основним інструментом для 500+ HVAC магазинів України впродовж року з МРР 150k+ ₴.

## 🎯 Market Positioning

### Target Customers

#### Сегмент 1: Малі HVAC магазини (50-500 товарів)
- **Кількість**: ~200 магазинів в Україні
- **Профіль**: Родинні бізнеси, локальні дилери Daikin, Systemair, Airwell
- **Проблеми**: 
  - Немає сучасної витрини (стара HTML, Flash сайты)
  - Ручне управління каталогом
  - Низька конверсія через складний процес замовлення
- **Рішення**: STROYKLIMAT дає професійну витрину за 30 хвилин
- **WTP**: 1500-2500 ₴/місяць

#### Сегмент 2: Середні HVAC дистрибьютори (500-5000 товарів)
- **Кількість**: ~100 компаній
- **Профіль**: B2B частково, завозять з Китаю, використовують Prom.ua
- **Проблеми**: 
  - Множинні каталоги на різних платформах
  - Неготові до бізнес-запитів (B2B RFQ)
  - Конкуренція від Prom.ua TopShops
- **Рішення**: Virtu showcase + експорт в Prom.ua XML
- **WTP**: 3000-5000 ₴/місяць

#### Сегмент 3: Встановлювачі / Інтегратори (100-500 товарів)
- **Кількість**: ~150 компаній
- **Профіль**: Компанії, що займаються монтажем, потребують витрину для запитів B2B
- **Проблеми**: 
  - FTP гоститинг невідповідний для B2B
  - Нижче конвертованих запит від клієнтів
  - Потребують інтеграції з CRM
- **Рішення**: STROYKLIMAT B2B версія з API для CRM
- **WTP**: 5000-8000 ₴/місяць

#### Сегмент 4: Дилити Маркетплейс (500-10000 товарів)
- **Кількість**: ~50 топ-дилера на Prom.ua
- **Профіль**: Професійні магазини, використовують рекламу, потребують конвертації
- **Проблеми**: 
  - Потребують окремої витрини для брендингу
  - З'єднання з Prom.ua API складне
  - Потребують аналітики по запитам цін
- **Рішення**: STROYKLIMAT Enterprise з аналітикою + API
- **WTP**: 10000-20000 ₴/місяць
