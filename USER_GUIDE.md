# 📖 STROYKLIMAT User Guide — Витрина HVAC для України

## Швидкий старт

### Для власників магазинів

#### 1️⃣ Адміністративна Панель (admin.html)

**Доступ**: http://stroyklimat.net/admin.html

**PIN**: 1234 (для Demo)

**Можете**:
- ✅ Завантажити каталог товарів
- ✅ Редагувати, видаляти, дублювати товари
- ✅ Управлювати категоріями
- ✅ Експортувати/Імпортувати каталог (JSON)
- ✅ Переглядати статистику
- ✅ Перемикати режим "Магазин" ↔ "Витрина"

**Як почати**:
1. Крокуйте https://stroyklimat.net/admin.html
2. Введіть PIN: **1234**
3. На панелі "Товари" натисніть "🔄 Вичитати каталог"
4. Натисніть "📝 Новий товар" і заповніть форму:
   - Назва
   - Ціна
   - Категорія
   - SKU
   - Марка
   - Статус наявності (В наявності / Під замовлення / Немає)
   - URL фото
   - Опис
   - Галочка "Виділені" (для топ-товарів)
5. Натисніть "💾 Зберегти" 
6. Натисніть "🚀 Публічна версія" (це опублікує каталог)

#### 2️⃣ Режим "Витрина" (Showcase Mode)

**Для Prom.ua дилерів, архітекторів, встановлювачів**

**Включає**:
- 💬 Кнопка "Запросити ціну" замість "Додати в кошик"
- 📧 Запит заповнює форму (ім'я, телефон, коментар)
- ✈️ Шеринг у Telegram, Viber, WhatsApp
- 📱 Автоматичне середання цінового запиту

**Як включити**:
1. Перейти на Admin > Налаштування
2. Нажимите на радіо-кнопку "📦 Витрина"
3. Натисніть "🚀 Опублікувати"

**Самих клієнтів ви помітите обновлення**:
- Кнопка "В кошик" → "💬 Запросити ціну"
- Страница cart.html → "Запити ціни" замість "Кошик"

**Як работает**:
1. Клієнт натискає "💬 Запросити ціну" на товарі
2. Заповнює форму: ім'я, телефон, коментар
3. Натискає "Надіслати"
4. Система пропонує поділитися через:

#### 3️⃣ Імпорт/Експорт Каталогу

**Экспортовать** (зробити резервну копію):
1. Admin > Імпорт/Експорт
2. Натисніть "📥 Експортувати каталог"
3. Завантажиться файл `catalog-backup.json`

**Імпортовать** (завантажити новий каталог):
1. Admin > Імпорт/Експорт
2. Перетягніть файл каталогу або натисніть "Оберіть файл"
3. Система покажає кількість нових/оновлених товарів
4. Натисніть "Прийняти"

**Формат JSON** (приклад):
```json
[
  {
    "id": "hvac-001",
    "title": "Вентилятор центробережный SYSTEMAIR",
    "price_uah": 4500,
    "category": "Вентиляція",
    "sku": "450-200",
    "brand": "SYSTEMAIR",
    "stock": "on_order",
    "images": ["https://example.com/img1.jpg"],
    "description": "Професійна вентиляція...",
    "featured": false
  }
]
```

### Для Клієнтів

#### 🛒 Стандартний Режим (Магазин)

**На головній сторінці**:
1. Пошук: "Введіть назву товару"
2. Категорійна навігація: Вентиляція, Кондиціонери, Опалення
3. На товарі:
   - Назва, фото, ціна
   - "В кошик" → додаєте в кошик
   - "☆ Додати в обране" → зберігаєте в обраному

**На сторінка кошика**:
- Список товарів з цінами
- Кнопки ± для зміни кількості
- Загальна сума
- "📞 Оформити по телефону"

**На сторінка обраного**:
- Збережені товари
- "Перейти в каталог" для поиска нових

---

#### 💬 Режим Витрина (Запрос Цены)

**На товарі**:
1. Замість "В кошик" бачите "💬 Запросити ціну"
2. Натискаєте кнопку
3. Заповнюєте форму:
   ```
   Ім'я: [введіть ім'я]
   Телефон: [введіть номер]
   Email: [опціонально]
   Коментар: [опціонально]
   ```
4. Натискаєте "Надіслати"
5. Система пропонує поділитися:
   - Telegram: пряме повідомлення з запитом
   - Viber: дзвінок/СМС
   - WhatsApp: чат
   - Email: письмо

**На сторінка кошика** (котра тепер "Запити Ціни"):
- Список запрошених товарів
- Напис "Запити до розробки: 5"
- "✈️ Написати на Telegram"

## 🐛 Часто розпитувані Питання

**Q: Як завантажити більше 1000 товарів?**
A: Адміністративна панель підтримує до 10,000 товарів. Для більше - потребує backend системи (SaaS).

**Q: Чи можна використовувати на своєму домені?**
A: Так! Просто скопіюйте папку `site/` на ваш хостинг (Vercel, Netlify, любий FTP)

**Q: Як інтегрувати з Prom.ua?**
A: Див. SAAS_STRATEGY.md - Prom.ua XML експорт розробляєтьс на етапі SaaS

**Q: Запити ціны зникли!**
A: Перевірте:
1. localStorage вручну не очищено (браузер > Налаштування > Кеш)
2. Режим зміни на "Магазин" (перевірте Admin > Налаштування)
3. Тест на іншому пристрої

**Q: Як передати каталог іншій людині?**
A: На Admin > Імпорт/Експорт натисніть "📥 Експортувати", поділіться JSON файлом
